# Python CSV to MySQL

CSV to SQL

### Requirements

Python3

Mysql Connector
```
pip install mysql-connector
```

## How to run

Change *mysqlConnector.py* params to your DB's params

```
mydb = mysql.connector.connect(
  host="IP or HOSTNAME",
  user="username",
  passwd="password"
)
```
In *docs/tables_queries.sql* you can check SQL schema
